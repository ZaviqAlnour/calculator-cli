# SIMPLE CALCULATOR

**A SIMPLE COMMAND-LINE CALCULATOR PROGRAM WRITTEN IN C**

THIS CALCULATOR SUPPORTS THE FOLLOWING OPERATIONS:

1. **ADDITION**  
2. **SUBTRACTION**  
3. **MULTIPLICATION**  
4. **DIVISION** (WITH ZERO-CHECK)  
5. **MODULUS** (INTEGER ONLY, WITH ZERO-CHECK)  
6. **POWER / EXPONENTIATION**  

THE CALCULATOR IS INTERACTIVE AND WILL CONTINUE RUNNING UNTIL THE USER CHOOSES TO EXIT.
